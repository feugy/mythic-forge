

Workstation Installation (Windows)
==================================

+ Instal NodeJS ([Download][1])

+ Instal MongoDB ([Download][2])

+ Instal Eclipse for JS Web developpers  ([Download][3])
  
  When written on your local drive, launched it.
  
  + Add the eGit eclipse plugin
    
  In the *Help* menu > *Install new software...*.  
  Adds a new site (eGit [http://download.eclipse.org/egit/updates]).  
  Select all items, and click on *Finished*
  
  + Instal Eclipse plugin for Node debugging ([Instructions][4])

    
    
[1]: http://nodejs.org/#download
[2]: http://www.mongodb.org/downloads
[3]: http://www.eclipse.org/downloads/packages/eclipse-ide-javascript-web-developers/indigosr2
